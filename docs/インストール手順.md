# インストール手順（各PC用・Windows）

このアプリ「出来高表」を、お使いのWindowsパソコンに入れる方法です。むずかしくありません。

---

## 1. インストーラを用意する

管理者から渡された **`出来高表_x.x.x_x64-setup.exe`** を、
社内ドライブ（Google Drive など）からダウンロードします。

## 2. インストールする

1. ダウンロードした **`.exe` ファイルをダブルクリック**します。
2. 「**WindowsによってPCが保護されました**」という青い画面が出たら：
   - **「詳細情報」** をクリック → **「実行」** ボタンを押します。
   - （社内で作った安全なアプリです。この表示は署名をしていないため出るだけです）
3. 画面の案内に従って「Next」「Install」を押すと、数十秒で完了します。
4. デスクトップとスタートメニューに **「出来高表」** のアイコンができます。

## 3. 使い始める

1. デスクトップの **「出来高表」** をダブルクリックして起動します。
2. **ログイン画面**が出ます。管理者からもらった **ログインID と パスワード** を入力。
3. 以上です。ホーム画面の大きなボタンから操作してください。

---

## よくある質問

**Q. ログインIDやパスワードがわからない**
→ 管理者に確認してください。管理者が「設定 → ユーザー」で発行します。

**Q. 別のパソコンで入力したデータが見えますか？**
→ はい。データはサーバーに保存されるので、どのPCからでも同じ内容が見えます。

**Q. インターネットは必要ですか？**
→ はい。常時インターネットに接続して使ってください。

**Q. 印刷したい / PDFにしたい**
→ 「印刷する」→「印刷する / PDFで保存」を押し、表示された画面で
　 プリンター（紙に印刷）または「Microsoft Print to PDF」（PDF保存）を選びます。

**Q. アップデート（新しい版）が来たら？**
→ 新しい `.exe` をもう一度ダブルクリックして上書きインストールするだけです。
